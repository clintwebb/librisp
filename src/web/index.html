librisp

Library to handle Reduced Instruction Set Protocols.

What is a Reduced Instruction Set Protocol

RISP is a binary protocol that is designed to be flexible and fast to process. It is different to a structure-based protocol in several key ways.

  1. 'Operations' are a group of simplistic commands.
  2. The 'Commands' that make up an operation can be presented in any order.
  3. An 'operation' is not performed until it receives an EXECUTE command.
